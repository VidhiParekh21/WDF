<h2>Project Description</h2>

<p>
StudentHub is a simple web portal developed for college students.
It provides one platform to access academic information such as
Attendance, Timetable, Assignments, Academic Calendar,
Announcements, Q&A Forum, and Contact Us.
</p>

<h2>Objectives</h2>

<ul>
<li>Provide one portal for students.</li>
<li>Easy access to academic information.</li>
<li>Simple and user-friendly interface.</li>
<li>Improve communication between students and college.</li>
</ul>
